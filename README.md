## Sobre o fluxo

- **Login:** Usuário e senha são validados (usuário: `admin` e senha: `123456`). Mensagem de erro aparece se for inválido.
- **Menu:** Permite navegação para cálculo de IMC, equipe ou voltar ao login.
- **IMC:** Permite entrada do nome, peso e altura, e mostra o resultado do IMC junto com o nome do usuário.
- **Equipe:** Mostra os integrantes do grupo e botão para voltar ao menu.
